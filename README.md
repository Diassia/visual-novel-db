## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:6006](http://localhost:6006) with your browser to see the result.

Homepage can be accessed on [http://localhost:6006/home](http://localhost:6006/home)
